# Eventia Core API

API REST para gestionar eventos, participantes y asistencia.

## Tecnologías
- Node.js + TypeScript
- Express
- TypeORM + Postgres
- Redis
- Jest + Supertest
- ESLint
- Docker + Docker Compose

## Requisitos
- Node 18+ / npm
- Docker (opcional)
- yarn / npm

## Instalación local
1. copiar `.env.example` a `.env` y ajustar valores.
2. instalar dependencias:
   ```bash
   npm ci
